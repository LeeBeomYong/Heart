## :🐶: Description
하루 평균 약 300마리의 동물이 버려지고 있습니다.
도움의 손길이 필요한 유기동물에게 새로운 삶을 선물할 수 있는 유기동물 매칭 플랫폼, HeartPet(하트펫)입니다.
HeartPet에서는 유기동물 입소 및 입양 그리고 후원할 수 있습니다.
